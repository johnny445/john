What is Proxy and Reverse Proxy?:
A proxy server is a go-between or intermediary server that forwards requests for content from multiple 
clients to different servers across the Internet. ... A reverse proxy provides an additional level of 
abstraction and control to ensure the smooth flow of network traffic between clients and servers.
